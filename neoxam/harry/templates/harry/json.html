Upload JSON Untranslated

On Delia Runtime installation, generate an aggregated JSON file from all untranslated files with delia-untranslated tool to submit all new labels to translation.

$ delia-untranslated --help
Usage:
    Usage: delia-untranslated [options]


Options:
  -h, --help            show this help message and exit
  -d SOURCE_DIR, --untranslated-directory=SOURCE_DIR
                        Specify the directory where the untranslated files are
                        stored.
  -o DEST_FILE, --json-file=DEST_FILE
                        Specify the output JSON file, if exists, it will be
                        consolidated.

Sample:

$ delia-untranslated -d $DELIA_GPDIR/work/untranslated -o /tmp/untranslated.json
